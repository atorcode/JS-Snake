Snake is a popular game in which the user controls a snake-like line and moves it across the game board in order to eat food and grow longer. The object of the game is to become as long as possible, or equivalently, to earn the highest score. To that end, the user must take care to maneuver the snake in a way that avoids collision with itself or with the outer walls. Unfortunately, your browser does not support canvas, which is needed in order for this game to run.
